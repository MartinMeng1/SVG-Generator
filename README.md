  # <SVG logo Generator>

  ## Description

  1. When the user is aksed about the color of the SVG logo.
  2. The user is able to enter a desired color.
  3. When the user is asked about the shape of the SVG logo.
  4. The user is able to choose a shape from the provided list.
  5. When the user is asked about the color of the SVG logo Shape.
  6. The user is ablt to enter a desired color of SVG shape.
  7. When the user is asked about the text for the logo within the limit of three characters.
  8. The user is able to enter the characters within the limit.
  9. A SVG file then will be generated.
